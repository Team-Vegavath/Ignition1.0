# Quick Asset Creation Guide

## Option 1: Create Simple Placeholder Assets (Recommended)

Create these files in the `assets` folder:

1. **icon.png** - 1024x1024 PNG (can be a solid color or simple logo)
2. **splash.png** - 1024x1024 PNG (can be same as icon)
3. **adaptive-icon.png** - 1024x1024 PNG (for Android)
4. **favicon.png** - 48x48 PNG (for web)

## Option 2: Use Online Tool

Go to: https://www.favicon-generator.org/
- Upload any image
- Generate all sizes
- Download and place in `assets` folder

## Option 3: Quick Fix - Use Solid Color Images

You can create simple solid color PNGs:
- Use any image editor (Paint, GIMP, online editor)
- Create 1024x1024 solid color images
- Save as PNG
- Name them: icon.png, splash.png, adaptive-icon.png
- Create 48x48 for favicon.png
